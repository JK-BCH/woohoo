# 격물치지(格物致知) — 어느 대학원생의 모험

쯔꾸루 감성의 브라우저 RPG. 안암의 석사과정 신입생 「연구」가 행정서류 슬라임과 카톡 알림 정령을 헤치고, 술의 힘으로 멘탈을 회복하며, 석사 디펜스와 일리노이 박사 예비심사(Prelim)에 도전합니다.

**▶ 플레이:** https://USERNAME.github.io/REPO/ ← 배포 후 주소로 교체

별도 설치 없이 브라우저에서 바로 실행됩니다 (PC·모바일, 크롬 권장).

## 게임 소개

- **스탯**: 멘탈(HP) · 논리(MP) · 논증력(ATK) · 철면피(DEF) · 마감력(SPD) · 통찰(LUK)
- **회복 수단**: 술 (소주·막걸리·폭탄주·버번·바롤로… 단, 폭탄주는 숙취 60%)
- **스킬**: 사서(四書)와 한중일 사상사에서 따온 12종 — 격물치지, 부동심, 술이부작, 사단지심, 치국평천하, 지행합일, 일이관지, 해체신서, 동도서기 등
- **제1장 안암**: 캠퍼스(낮/밤) · 문과대 채점 지옥 · 도서관 깊은 서고와 五經 골렘 · 술집 골목 회식 배틀 · 석사논문 3차 심사
- **제2장 일리노이**: UIUC 메인 쿼드(진저캣 올리 & 꽈이꽈이) · 아시아도서관 · 그린 스트리트(JK's Liquor 히든 재고) · 일리니 유니언 4층 던전(폭주 자판기 골렘) · 박사 예비심사 「So what?」
- **패배(현타)**: 소지금 50% 손실 + 아이템 일부 파손 — 무리하지 마세요. 현타는 비쌉니다.
