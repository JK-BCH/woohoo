# 포닥 생존 (STEM Postdoc Survivor)

이공계 박사후연구원 테마의 **서바이버형(탕탕특공대/Survivor.io 계열)** 게임.
단일 `index.html` — 바닐라 JS + Canvas, 외부 에셋 0 (스프라이트는 코드 픽셀아트, 사운드는 WebAudio 합성).
같은 리포의 격물치지 RPG(`/woohoo/`)와 독립적으로 `/woohoo/tangtang/` 에 배포된다.

## 플레이
- `index.html`을 브라우저로 열면 끝(서버 불필요).
- **이동**: WASD / 방향키 / 화면 드래그(모바일 터치 조이스틱). 사격은 **자동**(가까운 적, 보스 사거리 내면 보스 우선).
- 데이터(젬)를 모아 레벨업 → **업그레이드 3택**으로 빌드를 쌓는다.
- **테마 선택**(타이틀 메뉴) — 보스 라인업만 다름:
  - 🎉 **파티**: 병락(양식🍔)·지슬(한식🍚)·준식(라면🍜)
  - 🍻 **음주**: 수환(소주🍶)·자현(스카치🥃)·건호(버번🍾)
  - ⚽ **축구**: 주형(자동차🚗·느리고 큰 탄)·의중(축구공⚽·조준 연사)·준혁(축구공⚽·회전 탄막)
  - 9보스 **각자 다른 탄막 패턴**: 나선/십자/파도/양갈래/산탄/이중링꽃/자동차/조준/회전
- **보스 = 이벤트**: 보스전 동안 잡몹 스폰 정지(깔끔한 보스전). 첫 보스 ~90초, 이후 보스 사망 기점 90초 뒤 다음 보스. 처치 시 **자동 자석**(화면 젬 흡인)+리롤. HP 2000/3400/**6600**(최종 2배).
- 3번째가 **최종 보스**: 등장 시 모든 스폰 중단, 처치하면 **🎓 임용 성공 엔딩 + 통계**로 클리어.
- **중간보스 三目**: 보스 스케줄과 **무관하게 독립**(55초·이후 110초마다) 등장. **상목·중목·하목** 돌진 트리오, 잡으면 XP 듬뿍.
- **🧲 마그넷 아이템**: 잡몹이 가끔(~0.6%) 떨군다. 주우면 화면의 모든 경험치 젬을 빨아들여 한방에 레벨업. (보스는 경험치를 적게 줘 보스보다 잡몹 사냥이 성장의 축)
- **🎲 리롤**: 레벨업 업그레이드 선택 시 카드 3장을 다시 뽑는다. 기본 3회 + 보스 격파마다 +1.
- **사망 통계**: 죽으면 획득 스킬·레벨, 총 딜량/초당 딜량(DPS), 스킬별 딜 기여도(⭐최고 기여)를 보여준다.

## 콘텐츠
- **적 6종**: 피펫팅 잡무 · 밀린 이메일 · 데드라인 · Reviewer 2 · 리젝 레터 · PI 면담.
- **업그레이드 12종**: 임팩트 팩터/자동화 스크립트/공동 저자/메타 분석/카페인/워라밸/주말 수면/유레카/오픈 액세스/레퍼런스 매니저/공동연구자/학회 발표.
- **무기 진화(6종)**: 특정 업그레이드를 max까지 찍으면 해금
  - 자동화 스크립트→**프리프린트 폭격**(착탄 폭발) / 공동 저자→**대형 컨소시엄**(유도 추적) / 메타 분석→**체계적 리뷰**(관통 무제한)
  - 임팩트 팩터→**Nature 표지**(돌파구 확률↑) / 학회 발표→**기조 강연**(광역 둔화) / 공동연구자→**대형 연구실**(협력자 +2)
- **✦ 각성 시스템**: 무기를 max까지 올리면 **각성**(골든 충격파+플래시) → 그 무기 위력 **×2.4**.
  레벨 상한을 크게 높여서, 한 무기에 집중 투자하면 **기하급수적으로 강해지는** 빌드가 가능.
- **버프**: 논문 레이저(44뎀·쿨 2.5초·**3렙부터 ×2**)·예산 유도탄(34뎀·1.05초·**각성 시 광역 폭발**)·공동연구자(반경 92/132·**레벨당 ×2 곡선, 3렙≈×4**)·학회 발표(**전구간 ×3**). 고렙·집중 빌드일수록 화력이 가파르게 상승.

## 밸런스 (sim.js로 확정 — 수치 변경 시 반드시 동기화)
`node sim.js [runs]` — 전투 코어를 헤드리스로 포팅한 몬테카를로. "실전형" AI(외곽 선회 카이팅 + 우선순위 빌드).
플레이어 의도 조준이 없는 **보수적 하한**이므로, 실제 휴먼은 보스 격파율이 더 높다.

기준치(AI 하한, 마그넷 아이템·탄막 회피 미반영 → 실제 휴먼은 더 오래 생존):
| 지표 | 값 |
|---|---|
| 생존 중앙값 | ~140s (Lv19 도달) |
| 생존율 | 60s 100% · 120s 91% · 180s 24% |
| 보스 격파 | 1마리+ 43% · 2마리+ 10% |

버프 패치: 인용 연쇄(계수 26→34·쿨 1.5초·타격 4+lv)·블랙홀(틱 7→11·쿨 4.5초·범위↑) 상향,
저랩 구간 몹 상향(피펫 hp18→28·이메일 8→13·데드라인 14→22·리뷰어 80→100, 근접뎀 소폭↑).

⚠️ 중요 버그수정: `e.frozen<=0` 가드가 `undefined<=0`(=false)로 평가돼 **원거리 몹이 발사를
안 하고 보스 패턴(방사/조준/돌진)도 전혀 실행 안 되던** 문제 수정(`!(e.frozen>0)`). 이후 원딜을
위협이되 치명적이지 않게(주기 3초·탄속 160·뎀 60%·비중 22%) 재조정.

보스: HP 1600/2400/3300(×시간 스케일) + **3패턴 순환**(방사 탄막 → 조준 5연사 → 돌진).
적 탄환은 글로우+흰 코어로 어두운 배경에서도 잘 보이게, 원거리 몹(리젝/PI)은 20초부터 등장(부채꼴 2~3발).

밸런스 버프 패치: 화염 원툴 의존을 깨려 **평타(8→13뎀·연사 0.36초)·인용 연쇄(계수 16→26·타격 3+lv)** 상향,
보정으로 근접 몹 데미지·적 HP 스케일(0.015)·원딜 투사체 수를 함께 올림.

설계 의도: 초·중반(~3분)은 화려한 스킬로 쾌적하게 쓸어담는 **파워 판타지**,
180초 이후 적 HP·속도·물량이 급증해 5분 즈음 **결국 압도당하는** 서바이버 곡선.

핵심 튜닝 결정(시뮬이 잡아낸 것):
1. **레벨 스노우볼이 전부** — XP 곡선(시작 3, ×1.16+2)·수집범위(120)를 키워 빌드가 제때 올라오게.
2. **기본 관통 1** — 단일 타겟 탄으로는 군중 제어가 불가 → 모든 탄이 적 2기 관통.
3. **보스 우선 조준(520px)** — 자동조준이 늘 잡몹만 때려 보스가 안 죽던 문제 해결.
4. **보스전 잡몹 억제** + **첫 보스 120s로 지연** — "보스 도착 ~15초 후 압사" 패턴 완화.
5. **후반 가속 곡선** — 화염/빙결/미사일이 강력해 중반이 너무 쉬워지자, 180초 이후
   적 HP(+0.02/s)·속도(+0.0007/s)·스폰 물량(배치/캡↑)을 급증시켜 후반 벽을 복원.
