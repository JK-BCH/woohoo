# 격물치지(格物致知) — 어느 대학원생의 모험

쯔꾸루 감성의 브라우저 RPG. 안암의 석사과정 신입생 「연구」가 행정서류 슬라임과 카톡 알림 정령을 헤치고, 술의 힘으로 멘탈을 회복하며, 석사 디펜스 → 일리노이 박사 예비심사(Prelim) → 보스턴 Job Market까지 — **교수가 되는 그날까지** 달립니다. (v2.0으로 본편 3장 완결!)

**▶ 플레이:** https://jk-bch.github.io/woohoo/

별도 설치 없이 브라우저에서 바로 실행됩니다 (PC·모바일, 크롬 권장).

## 조작법

| 동작 | 키보드 | 모바일 |
|---|---|---|
| 이동 | 방향키 / WASD | 가상 십자패드 |
| 결정 · 조사 · 대화 | Z / Enter | A 버튼 |
| 취소 · 필드 메뉴 | X | B 버튼 / ☰ 메뉴 |

## 게임 소개

- **스탯**: 멘탈(HP) · 논리(MP) · 논증력(ATK) · 철면피(DEF) · 마감력(SPD) · 통찰(LUK)
- **회복 수단**: 술 (소주·막걸리·폭탄주·버번·바롤로·파피 밴 윙클… 단, 폭탄주는 숙취 60%)
- **스킬**: 사서(四書)와 한중일 사상사에서 따온 16종 — 격물치지, 부동심, 술이부작, 사단지심, 치국평천하, 지행합일, 일이관지, 해체신서, 동도서기, 수신제가, 박학심문, 법고창신, 활연관통 등 (Lv캡 35)
- **제1장 안암**: 캠퍼스(낮/밤) · 문과대 채점 지옥 · 도서관 깊은 서고와 五經 골렘 · 술집 골목 회식 배틀 · 석사논문 3차 심사
- **제2장 일리노이**: UIUC 메인 쿼드(진저캣 올리 & 꽈이꽈이) · 아시아도서관 · 그린 스트리트(JK's Liquor 히든 재고) · 일리니 유니언 4층 던전(폭주 자판기 골렘) · 박사 예비심사 「So what?」
- **제3장 보스턴 (신규!)**: 하버드-옌칭 포닥 생활 — 찰스 리버 거위떼와 보스 구스 · MIT 무한복도의 축구망령과 목(目) 삼형제 · 롱우드 의료가(과로·번아웃 디버프)와 BWH 건틀릿 · 요리 3인방의 합체요리(영구 스탯업)와 천상 라면 · 루카스의 보물 수집 · 논문 초고 집필 → **교수 임용 커미티 Job Talk** → [조교수] 엔딩
- **신규 디버프 6축**: 침묵 · 둔화 · 혼란 · MP 누수 · 과로 스택 · 기력 소진 — 해법은 수신제가와 천상 라면
- **패배(현타)**: 소지금 50% 손실 + 아이템 일부 파손 — 무리하지 마세요. 현타는 비쌉니다.

## 세이브

- 이 기기의 브라우저(localStorage)에 슬롯 3개로 저장됩니다.
- 브라우저 데이터 삭제, 시크릿 모드, 기기 변경 시 세이브가 사라질 수 있습니다.
  **메뉴 → 세이브/로드 → [세이브 코드 내보내기]** 로 코드를 복사해 백업해 두면 어디서든 복원할 수 있습니다.
- iOS Safari는 장기간(약 7일) 미접속 시 저장 데이터를 정리할 수 있으니, 코드 백업을 권장합니다.
