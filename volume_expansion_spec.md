## 1. 스토리 확장

### 1-1. 종신심사(Tenure) 편 — v3.0 본편 후속 【L · 최우선 서사】
- **입구**: 이미 구현된 intl2 `tenure` NPC(`S.tenureSeen`). 조교수 임용(S.boss3) 후 개방.
- **3막 구조 제안**:
  - 1막 *신임 조교수의 3중고*: 강의(강의평가 망령) · 연구(리비전 지옥) · 행정(서류 드래곤 — 1장 슬라임의 최종 진화형, 수미상관)
  - 2막 *6년의 시간*: "학기" 단위 진행 — 학기마다 업적 포인트(논문/강의/봉사)를 모으는 선택형 루프 (반복요소 2-1과 직결)
  - 3막 *Tenure Review*: 외부 추천서 3인(과거 인연 — 완당·JK·BWH 과장 재등장) → 종신심사 위원회 최종보스
- **신규 자산**: 맵 4~6종(교수 연구실·강의동·대학본부·학회장), 적 8~10종, 스킬 2~3종(Lv 35 캡 해제 → 40), 무기 1종
- **엔딩**: [종신교수] + "이제 자네가 완당이다" — 제자 NPC 등장으로 1장 오프닝과 수미상관
- **구현 노트**: 장 전환은 travelMenu 게이트 패턴, 최종보스는 BOSS_DEF 4번째 항목. Lv캡 해제는 NEXT_XP 연장 + gainLevels 상한만 수정. **세이브 v4 스냅샷 그대로 호환**.

### 1-2. 완당 선생 회상편 — "내가 하와이에 있을 때" 【M】
- 본편 내내 끊긴 떡밥의 회수. 옌칭 메일함에서 "낡은 일기장" 수령 → **플레이어블 회상**: 젊은 완당(전용 스프라이트)으로 하와이 디펜스 1전을 치르는 단편 (맵 1종 + 이벤트전 2~3개).
- 보상: 완당의 진심 서사 + 가보 아이템(예: 완당의 회중시계 — 전투 첫 턴 선공 보장).
- **구현 노트**: P 스탯 백업→교체→복원 패턴(이벤트 한정). startEventBattle 재사용. 규모 대비 서사 만족도가 가장 높은 항목.

### 1-3. 동료 후일담 사이드 퀘스트 3종 【M】
- **민서(동기)**: 박사 수료 후 진로 고민 — 업계행/학계행 상담 분기 대화 (보상: 통찰 +)
- **락·문·한 개별 의뢰**: 각자의 사연 1건씩 (락의 폐업 위기, 문의 보트 수리, 한의 영주권) — 요리 시스템 심화 연동 (신규 합체요리 3종 해금)
- **루카스의 기원**: 보스턴 5구역 단서 수집 → 루카스가 사실 전설의 ○○ — 방울 강화(디버프 무효 확률 10%)
- **구현 노트**: 전부 기존 npcTalk 분기 + S 플래그 패턴. 맵 신설 불필요.

### 1-4. 멀티 엔딩 분기 (학계 vs 업계) 【L · 보류 권장】
- Job Talk 직전 "테크기업 스카우트" 제안 분기 → 별도 엔딩. 서사 가치는 높으나 본편 정체성(교수가 되는 것)과 충돌 위험. Tenure 편 완성 후 NG+ 전용 분기로 재검토.

### 1-5. 플레이버 NPC 대사 3세트화 + 진행도 반응 【S】

## 2. 미니게임

> 공통 원칙: 기존 UI 자산(openMenu/bmsg/Canvas)을 재사용하는 **저비용 우선**. 신규 게임루프(MODE 추가)가 필요한 액션형은 1개만 엄선.

### 2-1. 찰스강 낚시 【M · 미니게임 최우선】
- **위치**: 찰스 보트하우스 옆 낚시터(신규 타일 1종 or NPC). 문(Moon)이 낚싯대 대여.
- **방식**: Canvas 게이지 바 타이밍 (MODE='fishing' 신설) — 움직이는 바를 적정 구간에서 Z. 3판 구성(입질→릴→끌어올리기).
- **보상 루프**: 물고기 12종(쓰레기~전설) → **물고기 도감**(수집요소 4-1 연동) + 문의 라면 재료 교환(잡어 3마리=재료B 할인) + 전설어 1종은 루카스 호감 이벤트.
- **구현 노트**: 게이지 로직 ~150줄. 물고기는 ITEMS가 아닌 S.fish={} 카운트로(인벤 오염 방지).

### 2-2. 체스 노인의 수읽기 문제 【S】
- hsq 체스 노인(기존 플레이버 NPC 승격): 일일 1회 "수읽기" 3지선다 (텍스트 퍼즐 풀 20문항 로테이션). 정답 시 ₩3,000 + 연승 보너스. openMenu만으로 구현.

### 2-3. JK의 블라인드 시음회 【S】
- 보스턴 JK: 주 1회(실제 날짜 기반) 시음 퀴즈 — 술 설명 듣고 이름 맞히기 (기존 15종 술 데이터 재사용). 3연속 정답 시 해당 술 1병. 술 도감(4-2)과 연동.

### 2-4. 채점 타임어택 【M】
- 1장 조교 퀘스트 확장: 제한 시간 내 O/X 연타(맞는 답안 Z, 틀린 답안 X). 점수제 + 최고기록 저장(S.gradeBest). 액션형이지만 입력이 단순해 기존 키 핸들러로 가능.

### 2-5. 거위 달리기 (보류) 【L】
- 강변 횡스크롤 회피 러너. 재미 잠재력은 높으나 별도 물리·충돌 루프 필요. 낚시 반응이 좋으면 차기 검토.

## 3. 반복요소 (엔드게임 루프)

### 3-1. 무한 던전 「리비전 지옥」 【M~L · 루프 최우선】
- **컨셉**: "Major Revision이 끝나지 않는다" — 같은 복도가 무한히 반복되는 차원(무한복도 끝 봉인문에서 진입, infinite 끝 보스 떡밥 회수).
- **구조**: 5층 단위 사이클. 층수에 비례해 기존 몹 스탯 스케일링(`mkE` 후 ×계수) + 5층마다 중간보스(기존 보스 강화판) + 보상 상자. 10층마다 탈출/계속 선택 (욕심 리스크).
- **보상**: 전용 화폐 "심사 포인트" → 교환소(영구 스탯·레어 술·수집품). 최고 도달 층수 기록(S.revBest).
- **구현 노트**: 맵 1종 재사용(매 층 동일 그리드 + 색조 변화), 인카운터 대신 고정 조우. 스케일링 공식은 시뮬로 층별 승률 곡선 확정(§8 방법론).

### 3-2. 보스 회상전 (명예의 전당) 【S~M】
- 본관 심사장에 "기억의 명패" — 격파한 모든 보스(석사심사~커미티~필드보스)를 **강화판으로 재전** (HP·ATK ×1.5, 보상은 골드·심사 포인트).
- 노피해 클리어 시 칭호(4-3 연동). BOSS_DEF/startEventBattle 전부 재사용이라 비용 낮음.

### 3-3. 일일 의뢰 로테이션 【M】
- 나이트 쉬프트(보스턴)·유학생회장(일리노이)·조교(안암) 3거점에 "오늘의 의뢰" — 실제 날짜(`new Date`) 시드로 일일 1건씩: 특정 몹 N마리 / 특정 지역 N승 / 아이템 배달.
- 보상: 골드 + 주간 누적 보너스(7일 도장 = 레어 술). **iOS 7일 정리 정책 고려해 코드 백업 안내와 묶을 것.**
- **구현 노트**: jkkills 카운터 패턴 + 날짜 문자열 비교. S 필드 3개 추가.

### 3-4. NG+ (회차 플레이) 【L · Tenure 이후】
- 엔딩 후 "안식년" — 1장으로 회귀하되 Lv/스킬/도감 유지, 적 전체 강화(×2.5), 회차 전용 대사(완당이 기시감을 느낌). 멀티 엔딩(1-4)의 그릇으로 활용. 세이브에 S.ngPlus 추가만.

## 4. 수집요소

### 4-1. 도감 시스템 (몬스터·술·물고기·요리) 【M · 수집 최우선】
- 필드 메뉴에 **「도감」 탭 신설** (fmView 패턴): 4분류.
  - **몬스터 도감**: 처치 시 등록(S.dex 객체 — victoryFinal/tallyBank 훅 1줄). 등록 시 약점·출몰지 표시 → 공략 정보로 기능.
  - **술 도감**: 마신 술 기록(playerAct item 훅). 15종 완집 보상: 칭호 [주당 박사].
  - **물고기 도감**: 낚시(2-1) 연동 12종.
  - **요리 도감**: 한의 합체요리 5종(+확장 3종).
- 완집률 % 표시 + 분류별 완집 보상. **세이브: S.dex/S.drank/S.fish/S.cooked 4필드 추가만.**

### 4-2. 기념 스탬프 랠리 【S~M】
- 전 28맵에 숨은 스탬프 포인트 1개씩(기존 H/조사 타일 패턴 — 게시판·기념비 등 장식 타일 재활용). 수집 현황은 지도 탭에 표시.
- 단계 보상: 9개(안암 완주)=막걸리 보따리 / 19개=SPD+3 / 28개 완집=**칭호 [전국구 대학원생]** + 텔레포트 권한(5-4 연동, "발도장 찍은 곳은 어디든").

### 4-3. 칭호 시스템 【S】
- 기존 서사 칭호([문학석사]~[조교수])를 데이터화(S.titles 배열) + 수집 칭호 추가(노피해 보스전·도감 완집·낚시왕·100승 등 12~15종).
- **장착 1개 = 소소한 패시브** (예: [주당 박사] 숙취 확률 절반, [전국구] 인카운터 -10%). 스탯 메뉴에서 교체. 밸런스 영향 작게(±10% 이내), 시뮬 기준치 침범 금지.

### 4-4. 전국 고양이 지도 【S】
- 올리·꽈이꽈이·루카스에 더해 **안암 골목 고양이 1·보스턴 뉴버리 고양이 1** 신설(랜덤 조우 패턴 재사용). 5묘 전부 친밀 달성 시 **고양이 소환 스킬**(전투 1회 한정 — 적 전체 혼란 부여, 연출용 필살기).

### 4-5. 레어 트로피 드랍 【S · 도감 의존】
- 보스·엘리트 몹 5% 드랍 장식품(골렘의 책장 파편, 구스의 왕관 깃털 등 10종). 효과 없음 — 순수 수집(도감 탭 전시). "한 번 더 잡을 이유"를 가장 싸게 만드는 장치.

## 7. 공통 구현 규약 (전 항목 적용)

1. **세이브**: 신규 상태는 전부 S 필드 **추가**로만 (init+def 동시, verify [8]). 객체형 필드(S.dex 등)는 `S.coffees` 패턴(빈 객체 default 병합) 준수.
2. **신규 적/NPC/맵**: §4-3 체크리스트 (DB+스프라이트+분기+verbs / NPCS+스프라이트+npcTalk / gen_maps+BFS+splice).
3. **전투 추가 시**: 몬테카를로 시뮬로 §8 기준치 곡선 확정 후 수치 고정. 기존 기준선(폭탄주 ₩12,000 등 경제 포함) 침범 금지.
4. **미니게임 신규 MODE**: 입력 핸들러의 MODE 분기에 명시 추가 + ESC/X 탈출 보장 (소프트락 방지).
5. **칭호/패시브 등 메타 보너스**: 단독 ±10% 이내, 중첩 시뮬 1회 필수.
6. 모든 패치 후: `node --check` → `verify.js` 0건 → (전투) 시뮬 → 버전 표기 → CHANGELOG.
