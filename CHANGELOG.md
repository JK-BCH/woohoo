## v2.9 — 교수동 증축과 보스턴 직행
- **인문대 교수동 2층 분리** (저렙 객사 방지): 1층은 인카운터 없는 안전 로비(연구실·행정 라운지·자판기), 강의 콘텐츠는 전부 2층 「줌 강의동」으로 이동(인카운터 1/16, 권장 Lv33+). 동쪽 끝 계단으로 왕복
  - 줌 강의실 3실 던전화 — 단상에 말 걸면 이벤트전, 각 방 고유 보상: A 첫 강의(질문 폭격 → ATK+5) · B 대형 교양(강의평가 정령 → DEF+5) · C 온라인 교수회의(마감 더미·회의 망령 → 최대 MP+20). A→B→C 순서 해금, 미션 [종장] 종강까지 버티기
- **교수동 진입로 신설**: 캠퍼스 y21 도로를 동쪽으로 연장해 교수동 정문까지 포장 — 더는 잔디(인카운터존)를 헤치고 출근하지 않는다
- **교수동 외관 차별화**: 사암 석조 + 청동 그린 지붕(테마5) — 캠퍼스의 적벽돌 건물들과 한눈에 구분, 내부도 석조 팔레트
- **타이틀 「보스턴 모드」 신설**: 1·2장을 건너뛰고 Lv20 박사로 3장(하버드 스퀘어)부터 시작 — 스킬 12종(Lv20까지 전부)·기계식 키보드·정착 장학금 ₩150,000·[문학석사]/[철학박사] 칭호 보유. 기존 세이브 무손상
- **스킬 계열 진화(접힘)**: 상위 스킬을 익히면 하위 호환 스킬이 전투 메뉴에서 자동으로 접힌다 — 지행합일→박학심문→활연관통 · 해체신서→활연관통 · 동도서기→법고창신 · 온고지신→수신제가. 메뉴 하단 「모든 스킬 보기」 토글로 언제든 전부 사용 가능(스킬 삭제·수치 변경·세이브 영향 없음). 만렙 기준 전투 메뉴 16→11종
- **죽은 스킬 3종 리워크** (몬테카를로 4,000회 재검증): ① 사단지심 — 4연타 타격마다 약화 1종 부여(ATK↓/약점/SPD↓/막기봉인, 2틱) = 유일한 디버프 살포기 ② 술이부작 — MP 5→3·반격 ×1.5→×2.0·반격 시 약점 노출, 단 **연속 사용 불가**(무한 반격 락 방지) ③ 경세치용 — 3턴→4턴·**술 버프와 합연산 중첩**(+50%까지)
- **필드보스 기믹 실구현 + 밸런스 재측정**: 보스 구스·스트레스 종양의 도전 화면이 경고하던 기믹(반격·과로 압박·고정뎀·격노)이 실제로는 코드에 없던 문제 수정 — 구스: 45% 반사 쪼기·3턴 날개 폭풍(둔화)·30% 둔화 돌진 / 종양: 4턴 고정뎀38+과로·35% 업무 떠넘기기(반격 불가)·반피 격노. 시뮬 확정: 무보급 구스 Lv30 33%/Lv32 97%, 종양 Lv30 35%/Lv32 82% (권장 Lv30-32 유지) · 보스별 카운터 테크: 구스↔사단지심(다단히트는 반격을 안 맞음) / 종양↔술이부작 위빙
- 메인(타이틀) 화면 정리: 챕터 소개 문단 삭제·조작법 한 줄 축약 — 버튼 4개와 함께도 화면 안에 들어온다
- 설계 문서: `skill_rework_proposal.md` — 스킬 16종 사용률 진단 + 리워크 방향 정리 (1차 계열 진화는 구현됨, 2차 리워크는 다음 사이클)
